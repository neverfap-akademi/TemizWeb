TemizWeb Filtresini Kur

Seçtiğiniz TemizWeb listesini uBlock Origin’e eklemek için aşağıdaki düğmeye basın.

TemizWeb Filtre seçimi hazırlanıyor.
uBlock Origin’e ekle uBlock Origin kurulu değilse yükle
Firefox ve uBlock Origin kurulu olmalı Bu sayfayı Firefox içinde açın.
“uBlock Origin’e ekle” düğmesine basın uBlock Origin abonelik onay ekranını açacaktır.
Aboneliği onaylayın Liste daha sonra otomatik olarak güncellenir.
Düğmeye bastığınızda hiçbir şey olmazsa uBlock Origin’in kurulu olduğunu ve bu sayfanın Firefox içinde açık olduğunu kontrol edin.

TemizWeb SafeSearch veya YouTube Kısıtlı Mod zorlamaz.
